# GenAI Thought Process Journal: Grounded System Prompts

Use this after the lesson.

1. What should the model be allowed to use as source of truth?
2. When should the assistant refuse instead of answering?
3. How will you prove citations support claims?
4. What conflict behavior does your domain need?
5. Which prompt regression test would catch the riskiest failure?

Mentor note: prompts are not decoration. In production RAG, prompts are behavior policy.
