# Chapter 2: The RAG Query Lifecycle

This chapter teaches what happens after a user asks a RAG assistant a question.
The learner follows the full production path:

1. User question.
2. Query understanding.
3. Retrieval query construction.
4. Metadata filtering.
5. Chunk retrieval.
6. Context assembly.
7. Grounded prompt construction.
8. Answer generation.
9. Citation and refusal behavior.
10. Evaluation, logging, and improvement.

The chapter keeps the explanation conversational, but the engineering standard
is production-grade. Learners should finish with a practical mental model they
can use in real systems, interviews, and architecture discussions.
