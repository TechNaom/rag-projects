Build a Traceable RAG Lifecycle Simulator

This project implements the lifecycle as a small, inspectable Python pipeline. The goal is not a fancy demo. The goal is a trace another engineer can debug.

starter.pysolution.py (reference)

1 Implement lifecycle stages

Classify intent, build retrieval query, apply filters, retrieve sources, answer or refuse, and emit a trace.

2 Add a golden eval

Prove the Germany refund question retrieves the EU policy, answers, and cites the expected source.

3 Extend one production scenario

Add security runbook, HR policy, developer docs, or contract-risk behavior.
