Exercises: Validate Metadata Records

Use starter.py to validate required metadata, normalize controlled values, and decide whether chunks can enter the index.

starter.pysolution.py

1 Check required fields

Validate identity, access, authority, and lifecycle metadata.

2 Normalize controlled values

Map inconsistent role, region, and authority values into allowed vocabulary.

3 Detect production risk

4 Recommend indexing decision

Return publish, review, block, or migrate_schema.
