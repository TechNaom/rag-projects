Project: Vector Index Release Planner

Build a release planner that validates vector records, recommends index strategy, and blocks unsafe promotions.

starter.pysolution.py

What you will build

  • A vector record validator.
  • A release-risk classifier.
  • A blue-green promotion recommendation.
  • A rollback and monitoring checklist.
  • A final release report for a retrieval platform team.
Product-level outcome

Index releases as engineering decisions

The project should say whether an index is ready, blocked, or conditional, and explain what must be fixed before traffic moves.
