# Thinking Journal: Source Documents

## Prompt to try

> You are auditing sources for a production RAG assistant. Given this source registry, identify which sources should be approved, quarantined, rejected, or reviewed. Explain the production risk and required metadata for each source.

## Reflection

1. Which source looked useful but was not authoritative?
2. Which source created the biggest access-control risk?
3. Which format had the highest extraction risk?
4. What metadata would make the source safer to retrieve?
5. How would you design synthetic data so learners see realistic failures?

## Builder note

Good RAG does not begin with embeddings. It begins with disciplined source judgment.
