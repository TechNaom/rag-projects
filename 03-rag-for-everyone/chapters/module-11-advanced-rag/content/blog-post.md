# Blog Post Draft — Advanced RAG

In this module, we break the topic into small decisions, concrete examples, and interview-ready trade-offs. By the end, the reader should know not only what each concept means, but when they would actually use it.

    ## What we cover

    - Agentic RAG
- Graph RAG
- Knowledge Graph RAG
- Multi-modal RAG
- SQL RAG
- Code RAG
- Long-context RAG
- Recursive retrieval
- Adaptive retrieval
- Multi-agent RAG
