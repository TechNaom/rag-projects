# Practice Bank: Types of RAG

For each scenario, choose a RAG type, justify the choice, name the production
risk, and define one eval.

## Scenarios

1. Internal FAQ over one handbook.
2. Country-specific refund assistant.
3. Security runbook assistant with escalation.
4. Legal contract assistant with clause relationships.
5. Research assistant over companies, people, and investments.
6. Support assistant that opens tickets.
7. Healthcare assistant over PDFs and medical images.
8. Developer docs assistant with tool execution.
9. Finance compliance assistant with refusal rules.
10. Sales assistant with quote approval workflow.
11. Incident assistant that needs runbook plus customer comms.
12. Product assistant over slides, tables, and text.
