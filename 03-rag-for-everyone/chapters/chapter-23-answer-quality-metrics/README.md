# Chapter 23: Answer Quality Metrics

This chapter teaches how to evaluate generated RAG answers after retrieval.

## Focus Areas

- Faithfulness
- Groundedness
- Answer relevance
- Citation correctness
- Completeness
- Refusal correctness
- Human review calibration
- Severity-based release gates

## Production Outcome

Learners build an answer quality scorer that can be reused in prompt, model, retriever, and corpus release checks.
