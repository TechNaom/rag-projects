Project: Answer Quality Scorer

Build a scorer that grades generated answers before release. This becomes the evaluation brain for product teams.

starter.pysolution.pyREADME

Production roadmap

  • Define a rubric: faithfulness, relevance, citation support, refusal correctness, completeness.
  • Implement deterministic checks first: missing citations, refusal keywords, unsupported banned terms.
  • Add human-review fields for high-risk cases.
  • Produce release gates: pass, warn, or block.
  • Store score traces by prompt, model, retriever, and corpus version.
