# Chapter 1 Exercises: Why RAG Exists

These exercises use only Chapter 1 concepts: model memory, trusted context,
citations, retrieval, and evaluation.

## Exercise 1 — Classify the question

Open `starter.py`. For each sample question, classify it as:

- `model_memory`
- `web_search`
- `rag`

Use `rag` when the question needs private, trusted, or source-cited context.

## Exercise 2 — Explain your reasoning

For each `rag` question, write one short reason explaining why model memory is
not enough.

## Exercise 3 — Add one policy question

Add one new question that would fit the Northkeep policy assistant.

## Checking your work

Run:

```bash
python3 starter.py
python3 solution.py
```

Your wording does not need to match the solution exactly. The important part is
that you can defend the classification.
