# Project: Production RAG Capstone Planner

Choose one capstone track and generate a milestone backlog. Full specs, required-corpus notes, and chapter-mapped roadmap checklists for every project below live in [`docs/production-and-capstone-projects.md`](../../../docs/production-and-capstone-projects.md).

## 5 Capstone Tracks

Pick one. Each is scoped to prove full curriculum mastery — completing any one exercises the whole pipeline, from source scoping (Ch.5-8) through deployment (Ch.27).

1. **Policy Intelligence Desk** (Enterprise Policy RAG) — employee/HR/compliance user, policy answer with audit trail.
2. **Incident Runbook Copilot** — on-call engineer, incident checklist and post-incident notes.
3. **Research Knowledge Studio** (Research Briefing Studio) — analyst/student/researcher, evidence board and exportable brief.
4. **Legal Contract Explorer** — legal ops/procurement, clause comparison and review queue.
5. **Customer Support Knowledge Hub** — support lead/agent, answer draft, ticket pattern, knowledge gap.

## 3 Production-Grade Projects

For after the capstone track — broader, resume-grade builds meant to prove you can ship something a company could actually run, not just complete an exercise.

1. **Multi-Tenant Documentation Copilot** — hard tenant isolation across shared infrastructure.
2. **Audit-Grade Compliance RAG** — sentence-level, signed evidence traceability for an external auditor.
3. **Real-Time On-Call Ops Copilot** — hard latency budget over a live-updating index.

See [`docs/production-and-capstone-projects.md`](../../../docs/production-and-capstone-projects.md) for the full brief on each: problem statement, required corpus, the specific chapters' techniques each must apply, and the deliverable artifact.
