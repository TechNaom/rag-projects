# GenAI Thought Process Journal: Context Injection

Use this journal after the lesson.

1. What evidence should the model see first, and why?
2. Which chunks should be dropped even if retrieval ranked them highly?
3. How will you prevent retrieved documents from becoming instructions?
4. What should your system do if two sources conflict?
5. Which context-injection metric would you track first in production?

Mentor note: context injection is where your architecture becomes visible to the model. Design it with the same care you give databases and APIs.
