What this is

A self-paced RAG course built as small, cross-linked HTML learning units. Every module keeps the same shell: lesson, quiz, interview prep, practice, exercises, lab, notebook, flashcards, and project.

What makes it useful

You study retrieval the way real teams experience it: hallucinations, noisy context, poor recall, weak evaluation, production constraints, and the design choices needed to respond.

What this is

A course shell that behaves like a product, not a folder of notes.

Each module is built to feel like a compact learning system: a real-world hook, plain-language explanations, quick recaps, checkpoint quizzes, practice, interview prep, and something you can build or defend.

Lesson-first flow

Start with the idea, move into the quiz, then deepen with practice, labs, notebooks, and the project without leaving the same module shell.

Instant browser feedback

Quiz completion is tracked locally in your browser. There is no backend, no sign-up, and nothing leaves your machine.

Production framing

The copy is written around failure modes, trade-offs, and evidence so you can explain a retrieval stack like someone who has to defend it.

Running locally

What to open, and when to run Python.

This repository has both a static course site and a separate Python RAG CLI project. The course does not use app.py. The Python entry point lives in the policy RAG project.

Open the course directly

For the homepage, lessons, quizzes, practice pages, labs, notebooks, and project pages, you can open the HTML files directly in a browser.

03-rag-for-everyone/index.html

Use a local server for the best experience

The styled markdown viewer needs HTTP to render README-style documents in-page. If you want the full shell, serve this folder locally.

cd 03-rag-for-everyone
python3 -m http.server 8000

Run the Python RAG CLI separately

The runnable Python app in this repo is the policy assistant project, not this course site.

cd 01-policy-rag-poc
pip install -r requirements.txt
python3 src/vectorstore.py
python3 src/main.py
