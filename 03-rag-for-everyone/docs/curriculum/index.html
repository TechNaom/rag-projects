Roadmap

The course mirrors the Python for Everyone architecture: chapters grouped into modules, each chapter ending with exercises, practice, interview prep, a project increment, and a GenAI thought-process journal.

Open RAG for Everyone Live UI

Module 1: RAG Foundations

Why RAG exists, the query lifecycle, RAG architecture families, and how to read system diagrams.

Outcome: explain a RAG system end to end.

Module 2: Documents and Ingestion

Source documents, parsing, cleaning, metadata, and repeatable ingestion pipelines.

Outcome: turn messy source material into traceable knowledge units.

Module 3: Chunking

Fixed, recursive, semantic, and hierarchical chunking, plus failure analysis.

Outcome: choose chunking strategies based on retrieval behavior.

Module 4: Embeddings and Similarity

Vectors, cosine similarity, local TF-IDF + SVD, neural embeddings, drift, and reindexing.

Outcome: reason about embedding choices and limitations.

Module 5: Vector Stores and Retrieval

ChromaDB, top-k retrieval, hybrid search, BM25, reranking, and retrieval trade-offs.

Outcome: design and debug retrieval stacks.

Module 6: Prompting for RAG

Context injection, grounded system prompts, citations, refusal behavior, and prompt testing.

Outcome: write prompts that are grounded and testable.

Module 7: Evaluation

Recall@K, MRR, faithfulness, groundedness, answer relevance, and failure analysis.

Outcome: measure RAG behavior instead of demoing by vibes.

Module 8: Production RAG

Access control, auditability, observability, cost, deployment patterns, and capstone design.

Outcome: defend a production RAG architecture and portfolio project.

Every Chapter Includes

Lesson, quiz, exercises, practice, interview prep, project, and GenAI thought-process journal.

Outcome: learn like a responsible new-age AI builder.
