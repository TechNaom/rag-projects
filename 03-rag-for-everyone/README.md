# RAG for Everyone

A book-like, example-driven RAG course built inside this repository using the same chapter-first pattern used in **python-for-everyone**.

## What is included

- Interactive HTML lessons
- Fill-in-the-blank quizzes
- Interview preparation pages
- Hands-on labs, exercises, practice banks, and projects
- Master chapters plus speaker notes, blog drafts, newsletters, cheat sheets, and workshop material
- Slide outlines, flashcards, architecture diagrams, notebooks, YouTube scripts, and LinkedIn carousel drafts

