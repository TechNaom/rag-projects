## Modules

1. **Module 1 — Foundation** (Why RAG, hallucinations, context windows)
2. **Module 2 — Data Ingestion** (PDFs, databases, APIs, preprocessing)
3. **Module 3 — Chunking Deep Dive** (Fixed, recursive, semantic, hierarchical)
4. **Module 4 — Embeddings** (Vector spaces, models, fine-tuning, drift)
5. **Module 5 — Vector Databases** (HNSW, IVF, filtering, sharding)
6. **Module 6 — Retrieval** (Similarity, BM25, RRF, multi-query, reranking)
7. **Module 7 — Re-ranking** (Cross-encoders, late interaction, trade-offs)
8. **Module 8 — Prompt Engineering** (Context injection, citation, guardrails)
9. **Module 9 — Evaluation** (Metrics, Ragas, DeepEval, custom evals)
10. **Module 10 — Production RAG** (Caching, streaming, monitoring, CI/CD)
11. **Module 11 — Advanced RAG** (Agentic, Graph, SQL, Long-context)
12. **Module 12 — Capstone Projects** (Full-stack systems, enterprise RAG)

## Learning Paths

### Path 1: Foundations First (Recommended for beginners)
1. Read Master Chapter
2. Watch YouTube script (mentally or via video)
3. Run notebook examples
4. Complete exercises
5. Answer interview questions
6. Build project

### Path 2: Hands-On First (For experienced engineers)
1. Run notebook examples
2. Complete hands-on lab
3. Read cheat sheet
4. Review interview questions
5. Deep-dive with blog post if needed

### Path 3: System Design Focus (For interviewers & architects)
1. Review architecture diagrams
2. Study comparison matrices
3. Answer interview questions
4. Design trade-offs section
5. Read speaker notes for systems thinking
